A simple CLI task tracker with zero dependencies.

## Install

```bash
go install github.com/zarldev/tsk/cmd/tsk@latest
```

## Usage

```bash
tsk add "buy milk"       # add a task
tsk list                 # show all tasks
tsk done 1               # mark task 1 complete
tsk rm 1                 # remove task 1
```
