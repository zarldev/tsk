tsk

task tracking for people who live in the terminal

$ go install github.com/zarldev/tsk/cmd/tsk@latest

$ cat demo.txt

$ tsk add "buy milk"
added task 1: buy milk

$ tsk list
  1 [ ] buy milk  (just now)

$ tsk done 1
task 1 marked done

$ tsk list
  1 [x] buy milk  (2m ago)

$ tsk list --pending
no tasks

$ tsk rm 1
task 1 removed

$ tsk --features

  • instant — add, complete, and remove tasks in seconds. no workflow to learn.
  • zero dependencies — single Go binary. no database, no config file, no runtime.
  • plain json — tasks live in ~/.tasks.json. read them, script them, back them up.
  • filtered views — list everything, or show only pending or completed tasks.

$ man tsk → read the docs
